Stop Throwing Away Your Decoder

Latent-XAI extracts the learnt local coordinate system of a VAE decoder using the Connection Laplacian — turning the "black box" of generative modelling into a transparent geometric map.

🚀 Get Started 📦 Install 📖 API Docs

Abstract

Variational Autoencoders (VAEs) learn a low-dimensional latent representation of high-dimensional data via an encoder φ: ΩX → Λ and a decoder ψ: ΩZ → ΩX, optimising a β-weighted ELBO comprising a reconstruction loss and a KL-divergence regulariser. While VAEs have driven discoveries in single-cell biology, molecular dynamics, and clinical data analysis, explanations of the learnt latent clusters with respect to input features remain poorly studied. Existing XAI approaches for VAEs operate on the encoder — measuring point-wise sensitivity of latent variables to ambient features — but this is largely uninformative when the latent space does not correspond to the intrinsic coordinate directions of the data manifold.

We propose Latent-XAI, a post-hoc geometric framework built on the key observation that a well-trained decoder ψ is a geometric repository: it encodes the full structure of the data manifold in its weights, regardless of how distorted the encoding may be. Rather than relying on local PCA — which is sensitive to sampling noise — Latent-XAI computes the Jacobian Ji = ∇zψ(z)|z=zi ∈ ℝD×L at each latent point and extracts an orthonormal tangent basis Ti ∈ ℝD×H via SVD, where H ≪ L is the intrinsic dimensionality of the manifold.

These decoder-derived tangent frames are aligned across a weighted k-nearest-neighbour graph using the discrete Connection Laplacian B (an NH×NH sparse block matrix), whose off-diagonal blocks Bij = −2wijQij encode the parallel transport map Qij = Ti⊤Tj between neighbouring tangent spaces. Minimising the Connection Laplacian's Rayleigh quotient over smooth vector fields reduces to a sparse eigenvalue problem, whose smallest eigenvectors yield globally coherent parallel vector fields — the smoothest, most fundamental directions of variation on the manifold — serving as intrinsic local coordinate charts in the ambient space ℝD.

This framework advances Global XAI: instead of asking "which features are most sensitive at a single point?", Latent-XAI asks "what are the dominant, linearly independent directions that characterise the neighbourhood of this point on the data manifold?" The method is validated across four datasets of increasing complexity — the Swiss Roll (flat 2D manifold), Twin Peaks (high-curvature surface), Alanine Dipeptide (66D molecular dynamics), and a pancreatic scRNA-seq dataset (3,000 genes, ~2,100 cells) — demonstrating recovery of intrinsic coordinates, curvature, free-energy potential wells, and branching differentiation lineages. Furthermore, because Latent-XAI requires only decoder evaluations and Jacobians on a given query set — not the original training data — it serves as a forensic tool for independent model verification under the EU AI Act (Articles 85–86), enabling unbiased post-hoc auditing of high-risk medical AI systems.

Bottom row: Given a neighbourhood of points, decoder-derived tangents build a discrete Connection Laplacian, whose eigenvectors characterise flows over the manifold.

Key Ideas

🎯

Decoder as Geometry

The VAE decoder encodes the data manifold's shape in its weights. Latent-XAI extracts it — don't throw it away!

∇

Jacobian Tangent Spaces

Compute J = ∇ψ(z) at each latent point. SVD extracts the H principal tangent directions without local PCA noise.

🌐

Connection Laplacian

Aligns local tangent frames across the neighbourhood graph to find globally smooth parallel vector fields.

🔍

Global XAI

Goes beyond point-wise sensitivity (SHAP/LRP) to neighbourhood-level manifold geometry — more informative and robust.

⚖️

EU AI Act Ready

Enables independent forensic auditing of high-risk medical AI systems without access to original training data.

🔧

sklearn-style API

Familiar fit() / fit_transform() interface. Works with any differentiable decoder.

Quick Start

Fit Latent-XAI to a trained VAE in three lines:

python 
from latentxai import lxai

# Z: latent codes (N, L), decoder: your trained VAE decoder (callable)
model = lxai(decoder=decoder, n_neighbors=20, sigma=0.3, principle_tangent_dims=2)
model.fit(Z)

# Access results
eigenvalues  = model.eigenvalues_       # (n_components,)
vector_fields= model.PVF_on_X_         # (n_components, N, D) — ambient space
vf_latent    = model.VF_on_Z_          # (N, L) — first VF in latent space
💡 Tip — sklearn compatibility

Latent-XAI follows scikit-learn conventions: hyperparameters in __init__(), fitted attributes end with _, and fit() returns self.
