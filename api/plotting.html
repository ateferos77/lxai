plotting

The plotting module provides vector field visualisation utilities and the GraphVelo class for projecting high-dimensional velocity fields onto 2D UMAP embeddings when the latent dimension exceeds 2.

ℹ️ Note — graphvelo dependency

GraphVelo.train() requires the optional graphvelo package. plot_flow() for 2D latent spaces works without it.

plot_flow

plot_flow(z_points, V_latent, overlay_color, nbrs_idx=None, umap_coords=None, skip=5, title="none", overlay_color_label="overlay", figsize=(10, 7), ref_pos=None, plot_title=True)

Plots the vector field as a quiver plot over a scatter of latent points. If the latent space has more than 2 dimensions, GraphVelo is used internally to project to the provided UMAP coordinates.

ParameterTypeDescription
z_pointsndarray (N, L)Latent coordinates. If L > 2, must also pass umap_coords and nbrs_idx.
V_latentndarray (N, L)Vector field in latent space (model.VF_on_Z_ for the first field).
overlay_colorndarray (N,)Values used to colour the scatter points (e.g. angle, time, cell type index).
nbrs_idxlist of arraysNeighbour indices (model.neighbor_indices_). Required when L > 2.
umap_coordsndarray (N, 2)2D UMAP embedding. Required when L > 2 to project arrows.
skipintPlot one arrow every skip points to avoid clutter. Default: 5.
titlestrPlot title (also used as filename: plots/{title}.png).
overlay_color_labelstrColorbar label for the scatter colour axis.
figsizetupleMatplotlib figure size. Default: (10, 7).
ref_posarray-like (2,), optionalIf provided, plots a white star marker at this (x, y) reference position (e.g. energy minimum).
plot_titleboolWhether to display the title on the plot. Default: True.
💡 Tip — Vector normalisation

Arrow lengths are unit-normalised before plotting: v_unit = v / (||v|| + ε). This ensures all arrows are the same visual length, emphasising direction rather than magnitude.

python 
from latentxai.plotting import plot_flow

# 2D latent space — direct quiver plot
plot_flow(
    z_points=Z,
    V_latent=model.VF_on_Z_,
    overlay_color=color_values,
    skip=10,
    title="swiss_roll_VF1",
    overlay_color_label="Swiss Roll position"
)

# 4D latent space — UMAP projection via GraphVelo
plot_flow(
    z_points=Z,
    V_latent=model.VF_on_Z_,
    overlay_color=cell_type_ids,
    nbrs_idx=model.neighbor_indices_,
    umap_coords=umap_embedding,
    skip=5,
    title="pancreas_VF1"
)

GraphVelo

class GraphVelo(X_data=None, V_data=None, nbrs_idx=None, gene_subset=None, approx=False, n_pcs=30, mo=False)

Manifold-constrained velocity projection. Used by plot_flow when the latent dimension is greater than 2 to project high-dimensional velocity vectors onto a 2D UMAP embedding.

train

gv.train(a=1, b=10, r=1, loss_func=None, ...)

Learns a transition matrix by minimising the tangent space projection (TSP) loss:

L(wᵢ) = a |v̂ − v|² − b cos(u, v̂) + λ Σⱼ |wᵢⱼ|²

where v̂ = Σⱼ wᵢⱼ · dᵢⱼ approximates the velocity from neighbour displacements.

project_velocity

gv.project_velocity(X_embedding, T=None) → ndarray (N, 2)

Projects trained velocity vectors onto a low-dimensional embedding using the learned transition matrix T.

tangent_space_projection

tangent_space_projection(X, V, C, nbrs, a=1.0, b=0.0, r=1.0, loss_func="linear", n_jobs=None)

Core function that learns a sparse transition matrix via the TSP loss. Runs in parallel across all data points using joblib.

Returns: ndarray (N, N) — dense transition matrix E with entries E[i, nbrs[i]] = wᵢ.
