Latent-XAI › API Reference › lxai

lxai

The lxai class is the core object of the Latent-XAI framework. It follows scikit-learn estimator conventions: hyperparameters are set in __init__(), fitted attributes end with an underscore (_), and fit() returns self.

class lxai(
    decoder='PCA',
    n_neighbors=20,
    n_components=5,
    principle_tangent_dims=3,
    sigma=0.1
)

Constructor Parameters

ParameterTypeDefaultDescription
decoder callable | 'PCA' 'PCA' The VAE decoder function. Must accept a torch.Tensor of shape (N, L) and return (N, D). If 'PCA', uses local PCA for tangent estimation (no GPU decoder required).
n_neighbors int 20 Number of nearest neighbours for the KNN graph construction in latent space. Larger values capture smoother but less local structure.
n_components int 5 Number of eigenvectors (parallel vector fields) to compute from the Connection Laplacian. Corresponds to M in the paper.
principle_tangent_dims int 3 Number of principal tangent directions (H) extracted from the Jacobian SVD. Must be ≤ latent dimension L. Corresponds to the intrinsic dimensionality of the manifold.
sigma float | 'auto' 0.1 Gaussian kernel bandwidth for edge weights. Set to 'auto' for self-tuning (Zelnik-Manor & Perona 2004), which uses per-point local scale.

Fitted Attributes

Available after calling fit():

AttributeShapeDescription
Z_ (N, L) Latent points used for fitting.
Jac_ (N, D, L) Jacobian matrices ∇ψ(z) at each latent point.
T_ (N, D, H) Orthonormal tangent space bases (top H left singular vectors of each Jacobian).
Jac_S_ (N, L) Normalised singular values of the Jacobian at each point.
eigenvalues_ (M,) Eigenvalues of the Connection Laplacian, sorted by absolute value. Small eigenvalues correspond to smooth parallel vector fields.
eigenvectors_raw_ (M, N, H) Raw eigenvectors in local tangent coordinates.
PVF_on_X_ (M, N, D) Parallel vector fields in ambient (data) space. The m-th field is PVF_on_X_[m].
VF_on_Z_ (N, L) First vector field projected to latent space via the Moore-Penrose pseudoinverse of J.
neighbor_indices_ list of arrays Neighbour indices for each point in the KNN graph.

Methods

fit

model.fit(Z, data=None, knn_distances=None, orient_v_0=None, orient_to=None, smooth_VF=False)

Fit the Latent-XAI model. Computes Jacobians, builds the Connection Laplacian, and extracts parallel vector fields.

ParameterTypeDescription
Zndarray (N, L)Latent codes. The input to the Connection Laplacian computation.
datandarray (N, D), optionalOriginal data in ambient space. Required when decoder='PCA'.
knn_distancessparse matrix, optionalPrecomputed KNN distance graph. If None, computed automatically from Z.
orient_v_0ndarray, optionalReference vectors for tangent space orientation alignment.
orient_tondarray, optionalReference vectors for vector field sign orientation (flip eigenvectors to align).
smooth_VFboolIf True, smooth vector fields by averaging over neighbours. Default: False.

Returns: self — fitted estimator.

python 
from latentxai import lxai

model = lxai(decoder=vae.decode, n_neighbors=20, sigma=0.3, principle_tangent_dims=2)
model.fit(Z)  # Z: ndarray (N, L) of latent codes

# Access results
print(model.eigenvalues_)       # (5,) eigenvalues
print(model.PVF_on_X_.shape)    # (5, N, D)

fit_transform

model.fit_transform(Z, **kwargs)

Convenience method: fit the model and return the first parallel vector field in ambient space.

Returns: ndarray (N, D) — first parallel vector field.

compute_decoder_jacobian

model.compute_decoder_jacobian(z, custom_decoder=None)

Computes the Jacobian matrix of the VAE decoder at each point using PyTorch autograd (backward() with one-hot outputs).

ParameterTypeDescription
ztorch.Tensor (N, L)Latent vectors on the same device as the decoder.
custom_decodercallable, optionalAlternative decoder function to use instead of self.decoder.

Returns: torch.Tensor (N, D, L) — Jacobian matrices. Detached from the computation graph.

project_fields_to_latent

model.project_fields_to_latent(z_points, V_ambient, precomputed_jacobians=None)

Projects a vector field from ambient space back to latent space using the Moore-Penrose pseudoinverse: v_latent = (JTJ)-1 JT v_ambient. Tikhonov regularisation (ε = 1e-6) is applied for numerical stability.

Returns: ndarray (N, L) — vector field in latent space.

save

model.save(filename)

Serialise the fitted model to a file using dill (supports lambda functions and PyTorch modules).

python 
model.save("latentxai_swissroll.pkl")

# Load later
import dill as pickle
with open("latentxai_swissroll.pkl", "rb") as f:
    model = pickle.load(f)

Internal Methods

The following methods are used internally but are accessible if needed:

MethodDescription
_build_connection_laplacian(T_z, weights_csr, N, H)Vectorised assembly of the sparse block Connection Laplacian matrix B of size NH×NH.
_compute_tangent_spaces(Jac, orient_v_0)Batched SVD of Jacobians to extract orthonormal tangent bases T_i of shape (N, D, H).
_PVF(Z, Jac, weights_csr, orient_v_0)Full parallel vector field computation: builds B, runs GPU eigsh, stores results.
_compute_weights(data_graph)Gaussian kernel weights: exp(−d²/2σ).
_compute_adaptive_weights(data_graph)Self-tuning Gaussian weights (Zelnik-Manor & Perona 2004).
compute_local_PCA(data, knn_graph, Z_dim_num)Local PCA fallback for tangent estimation when no decoder is available.
