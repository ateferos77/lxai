Installation

Latent-XAI is a Python library that requires PyTorch for Jacobian computation and CuPy for GPU-accelerated eigendecomposition. A CUDA-capable GPU is strongly recommended for large datasets.

⚠️ GPU Requirement

Latent-XAI uses cupy for sparse eigendecomposition (cupy.sparse.linalg.eigsh). A CUDA-capable NVIDIA GPU is required. The paper experiments were run on an NVIDIA H100 NVL GPU.

Install from Source

Clone the repository and install the package in development mode:

bash 
# Clone the repository
git clone https://github.com/yourusername/latentxai.git
cd latentxai

# Install in editable mode
pip install -e .

# Or install dependencies manually
pip install torch scipy scikit-learn pandas matplotlib tqdm dill

Installing CuPy (GPU)

CuPy must be installed matching your CUDA version. Check your CUDA version with nvcc --version or nvidia-smi.

bash 
# CUDA 12.x
pip install cupy-cuda12x

# CUDA 11.x
pip install cupy-cuda11x

# Verify CuPy installation
python -c "import cupy; print(cupy.__version__, cupy.cuda.runtime.runtimeGetVersion())"

Conda Environment (Recommended)

For a clean reproducible environment, we recommend conda:

bash 
conda create -n latentxai python=3.10
conda activate latentxai

# PyTorch with CUDA 12
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia

# Scientific stack
conda install numpy scipy scikit-learn pandas matplotlib tqdm
pip install dill cupy-cuda12x

Verify Installation

python 
import torch
import cupy as cp
from latentxai import lxai

# Check GPU availability
print(f"PyTorch CUDA: {torch.cuda.is_available()}")
print(f"CuPy devices: {cp.cuda.runtime.getDeviceCount()}")

# Create a minimal lxai object (decoder='PCA' for CPU-only testing)
model = lxai(decoder='PCA', n_neighbors=10)
print("✓ Latent-XAI imported successfully")
ℹ️ Note — CPU Fallback

Setting decoder='PCA' uses local PCA for tangent space estimation instead of the decoder Jacobian. This avoids the need for a GPU-accelerated decoder but still requires CuPy for the eigendecomposition step.
