🧪 Tutorial 3: Alanine Dipeptide

Molecular Dynamics Real-World Data 66D → 2D 11,215 samples

Alanine Dipeptide is a widely-used benchmark in molecular dynamics (MD). The protein consists of 22 atoms (66-dimensional position data) governed by two principal dihedral angles φ and ψ. This tutorial demonstrates that Latent-XAI can recover intrinsic geometry from real-world, high-dimensional noisy data.

Scientific Background

The configuration space of Alanine Dipeptide is topologically a torus (parameterised by φ and ψ). Because a simple Gaussian-prior VAE cannot represent a torus, we restrict analysis to states where φ ∈ (0, 2) radians, reducing the topology to a circle.

ℹ️ Why restrict φ?

The full dihedral angle space is a torus — topologically impossible to embed faithfully with a standard Gaussian prior VAE. By restricting to φ ∈ (0, 2), the topology becomes a circle, which the VAE can represent as a ring-shaped latent structure.

Data Preprocessing

python
import numpy as np
    import torch
    from sklearn.preprocessing import RobustScaler

    # Compute backbone dihedral angle from atom coordinates (matches notebook math).
    def dihedral(x: np.ndarray, i: int, j: int, k: int, l: int):
      e_ji = x[:, i] - x[:, j]
      e_jk = x[:, k] - x[:, j]
      e_kl = x[:, l] - x[:, k]
      u = np.cross(e_ji, e_jk)
      v = np.cross(e_kl, e_jk)
      w = np.cross(u, v)
      angle = lambda a, b: np.arccos(np.clip(np.einsum('...i,...i', a, b) / (np.linalg.norm(a, axis=-1) * np.linalg.norm(b, axis=-1)), -1, 1))
      alpha = angle(u, v)
      beta = np.where(np.linalg.norm(w, axis=-1) > 1e-6, angle(e_jk, w), 0)
      alpha[beta > 1e-3] *= -1
      return alpha

    # Remove translation/rotation so VAE learns internal conformational changes.
    def align_coordinates(coords):
      N, _, _ = coords.shape
      com = coords.mean(axis=1, keepdims=True)
      coords_centered = coords - com

      reference = coords_centered[0]
      aligned_coords = np.zeros_like(coords_centered)
      aligned_coords[0] = reference

      for i in range(1, N):
        current = coords_centered[i]
        matrix_h = current.T @ reference
        u, s, vt = np.linalg.svd(matrix_h)
        rotation_matrix = vt.T @ u.T
        if np.linalg.det(rotation_matrix) < 0:
          vt[-1, :] *= -1
          rotation_matrix = vt.T @ u.T
        aligned_coords[i] = current @ rotation_matrix.T
      return aligned_coords

    # Load MD trajectory used in the notebook.
    x = np.load('data/trj.npy').reshape(-1, 22, 3)
    E = np.load('data/nrg.npy')

    # Extract Ramachandran angles from atom indices used in the notebook.
    phi = dihedral(x, 4, 6, 8, 14)
    psi = dihedral(x, 6, 8, 14, 16)

    # Restrict topology to circular subset: phi in (0, 2).
    keep = (phi > 0) * (phi < 2)
    phi, psi, x, E = phi[keep], psi[keep], x[keep], E[keep]

    # Prepare VAE input: align -> flatten -> robust-scale -> torch tensor.
    x_a = align_coordinates(x)
    X_flat = x_a.reshape(x_a.shape[0], -1)  # (N, 22, 3) -> (N, 66)
    scaler = RobustScaler()
    X_scaled = scaler.fit_transform(X_flat)
    X_tensor = torch.tensor(X_scaled, dtype=torch.float32)

python
import torch
    import torch.nn as nn
    import torch.optim as optim
    from torch.utils.data import TensorDataset, DataLoader

    # Same architecture/hyperparameters as the notebook (66 -> 2 latent).
    class VAE(nn.Module):
      def __init__(self, input_dim=66, latent_dim=2, hidden_dim=512):
        super(VAE, self).__init__()
        self.encoder = nn.Sequential(
          nn.Linear(input_dim, hidden_dim),
          nn.LeakyReLU(),
          nn.Linear(hidden_dim, hidden_dim),
          nn.LeakyReLU(),
          nn.Linear(hidden_dim, hidden_dim),
          nn.LeakyReLU()
        )
        self.fc_mu = nn.Linear(hidden_dim, latent_dim)
        self.fc_logvar = nn.Linear(hidden_dim, latent_dim)

        self.decoder = nn.Sequential(
          nn.Linear(latent_dim, hidden_dim),
          nn.ELU(),
          nn.Linear(hidden_dim, hidden_dim),
          nn.ELU(),
          nn.Linear(hidden_dim, hidden_dim),
          nn.ELU(),
          nn.Linear(hidden_dim, input_dim)
        )

      def encode(self, x):
        h = self.encoder(x)
        return self.fc_mu(h), self.fc_logvar(h)

      def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

      def decode(self, z):
        return self.decoder(z)

      def forward(self, x):
        mu, logvar = self.encode(x)
        z = self.reparameterize(mu, logvar)
        recon_x = self.decode(z)
        return recon_x, mu, logvar

    # Notebook beta schedule: warmup -> ramp -> fixed beta=2.0.
    def train_alanine_vae(X_tensor, device):
      dataset = TensorDataset(X_tensor)
      dataloader = DataLoader(dataset, batch_size=256, shuffle=True)
      model = VAE(input_dim=66, latent_dim=2).to(device)
      optimizer = optim.Adam(model.parameters(), lr=1e-4)

      for epoch in range(500):
        if epoch < 50:
          beta = 0.0
        elif epoch < 250:
          beta = 2.0 * ((epoch - 50) / 200)
        else:
          beta = 2.0

        for batch in dataloader:
          x_batch = batch[0].to(device)
          optimizer.zero_grad()
          recon_x, mu, logvar = model(x_batch)
          mse = nn.functional.mse_loss(recon_x, x_batch, reduction='sum')
          kld = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp()) if beta != 0 else 0
          loss = mse + (beta * kld)
          loss.backward()
          optimizer.step()
      return model

import sys
    import torch
    from matplotlib.ticker import MaxNLocator

    sys.path.append("/scratch/htc/bzfsunka/latentxai/latentxai")
    import latentxai, plotting

    # Encode dataset into latent coordinates Z (matches notebook convention).
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    trained_model = train_alanine_vae(X_tensor, device)
    trained_model.eval()
    with torch.no_grad():
      Z = trained_model.encode(X_tensor.to(device))[0].cpu().numpy()

    # Fit Latent-XAI exactly with notebook settings.
    AD_lxai = latentxai.lxai(decoder=trained_model.decoder,
                 principle_tangent_dims=2,
                 n_neighbors=30,
                 n_components=20,
                 sigma=0.3)
    AD_lxai.fit(Z)

    # Inspect first 10 eigenvalues (same plotting intent as notebook).

Locating the Potential Well

The free energy well (white star in panels b, c, e, f) represents the most energetically stable configuration of the peptide. Crucially:

  • In the latent space, the potential well (white star) did not correspond to the densest part of the points — demonstrating that the encoder removes local geometry.
  • Visualizing the first eigenvector, we observed that the vector field captures the rotation in the direction of ψ.
  • Subsequent eigenvectors produced vector fields with saddle-point-like behavior in close proximity to the free energy minima.
  • The fifth eigenvector represented a spiral vector field near the free energy minima, possibly capturing the deformations involved in the transition into the well.

💡 Key insight

The encoder discards local geometry (the energy minimum is not at the densest latent point), but the decoder preserves it. Latent-XAI uses the decoder to re-expose this information through the saddle/spiral patterns in the higher eigenvectors.
